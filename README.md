# ATM Smartcard Simulator

A Python-based ATM simulation system with **admin management and smartcard authentication**.

## Features

* Customer login
* Withdraw and deposit money
* Balance inquiry
* Transaction history
* Admin management panel
* Smartcard authentication system

## Technologies Used

* Python
* JSON database
* Smartcard ATR verification

## Project Structure

```
atm_admin/
│
├── main.py
├── run_atm.bat
├── atm_app/
│   ├── account.py
│   ├── admin.py
│   ├── customer.py
│   ├── utils.py
│
├── data/
│   ├── accounts.json
│   └── transactions.json
```

## How to Run

1. Install Python
2. Run:

```
python main.py
```

or

```
run_atm.bat
```
